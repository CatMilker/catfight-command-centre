# Catfight Command Centre

## What's in it

- BHG grind guide with fast track (VLRT → ERT), cert chain confirmed from 4.8 live game files
- Cargo, mining and Nyx grind guides — tier by tier from zero aUEC
- Xenothreat phase guide with Catfight role split
- Rep tracker — BHG, Covalex, People's Alliance, Foxwell, Headhunters, Shubin, InterSec
- Session income estimator across 17 activities
- Org ops role planner
- New player day-one checklist
- System navigator — Stanton, Pyro, Nyx
- Blueprint roadmap by role
- Mission reference — all 9 factions with full difficulty tables
- Blueprint sources — 21 confirmed drop sources from 4.8 live files
- Minerals panel — 35 minerals, confirmed locations
- Loadout builder — 8 ships, multiple presets, buy route planner
- Ship database — 275+ ships

## Data sources

All data extracted from Star Citizen 4.8 live build (4.8.0.11825000):

- `global.ini` — item names, component names, mineral locations, planet descriptions
- `all_sc_data.txt` — mission data, blueprint drop tables, difficulty ratings
- `ships_desc_enhancements.ini` — ship stats (crew, role, SCM speed, armor)
- `Game2.dcb` — BHG rank chain, cert unlock structure

Ship prices pending 4.8 live UEX data — currently showing 4.7.2 values where available.
