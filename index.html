FASTEST PATH TO ERT
Step 1 — Accept and complete the VLRT Certification mission from the BHG contract board. Unlocks at Probation rank (starting rank). 50 XP reward.
Step 2 — Grind VLRTs until you hit Junior rank. Complete the LRT Cert mission. 50 XP.
Step 3 — Grind LRTs → hit Guild Member rank. Complete MRT Cert. Then immediately do the HRT Cert — this is the biggest income jump. 50 XP each.
Step 4 — HRTs give enough aUEC to buy a proper fighter. Grind HRTs → Senior rank → VHRT Cert → grind VHRTs → Veteran rank → ERT Cert.
Key tip — Cert missions are one-time only. Complete each immediately when the rank unlocks — don't skip. Each cert gate requires a specific mission to be completed, not just the rep threshold.
Confirmed from 4.8 live DCB: rank names are Not Eligible → Probation → Applicant → Junior → Member → Guild Member → Senior → Veteran → Master. Actual XP thresholds per rank are server-side only — not in game files. Track progress in-game via mobiGlas → Reputation.

TIER 4
Xenothreat Event — Org Ops
Event rewards + CDF rep + blueprint drops
▼
ORG PLAYHIGH RISK 7/7TIME-LIMITED EVENT
Xenothreat is a phased system-wide event defending Stanton. CDF reputation stacks on top of aUEC payouts. Priority content for Catfight — coordinate sessions around active event windows. Confirmed back in 4.8.
EVENT PHASES
PH1
Ship Combat — Blockade defense. Xenothreat fleet equipped with quantum jammers blockades stations and disables defensive turrets. Fighters engage and hold the line. CDF volunteers are active — anyone can contribute. Kill Xenothreat ships in the area for bounty XP (+16,000 XP per confirmed target, −8,000 XP failure penalty). Stick with the fleet, don't lone-wolf — they spawn in packs and overwhelm solo fighters.
PH2
Zeta-Prolanide Recovery — Cargo run. Xenothreat steals Zeta-Prolanide from station reserves. Scattered across nearby space in clusters — higher outlaw presence around larger clusters. Bring a tractor beam. Recover and return to station freight elevator before the material degrades. Regular bonus payments for combined org effort. Best role for cargo ships and haulers who can't compete in the fighter phase.
PH3
FPS Boarding Defense. Xenothreat boards stations — defend against on-foot attackers. Gear up in medium or heavy armor before entering. Clear each section methodically — Xenothreat displays unprovoked aggression against security personnel and will not retreat. Ship heist missions also run during this phase where Xenothreat cargo ships are locked in asteroid hangars — these have up to 1,160 enemies swarming the area, org-only content.
BP
Blueprint drops. Xenothreat bounty missions have a confirmed blueprint reward system. Specific drops not fully catalogued yet — verify in-game when event activates. Historical drops include weapon and armor blueprints. Stack event contracts alongside the CDF volunteer work for maximum returns per session.
CATFIGHT ROLE SPLIT
✈
Fighters (2–4 pilots): F7C Mk II, Sabre, Vanguard. Focus phase 1 ship combat. Stick to groups, don't chase strays outside the zone.
📦
Cargo runner (1–2 pilots): Cutlass Black, C2 Hercules, Zeus CL. Phase 2 Zeta-Prolanide recovery. Bring tractor beam. High volume = bonus payments.
🏥
Medical/support (1 pilot): Cutlass Red. Keeps fighters alive between respawns. Park near the action, don't push into combat. Critical for multi-hour sessions.
🔫
FPS team (2+ members): Phase 3 boarding defense. Medium or heavy armor — Palatino, Monde, ORC-mkX. Coordinate entry points, don't split up inside the station.
Xenothreat ship heist missions (asteroid hangar phase) list up to 1,160 enemies — confirmed from game files. Do not attempt solo. Org fleet only.
TIP: Event windows are time-limited. Check Spectrum or the CDF mission board for active status. Set up an org Discord ping for event activation — missing the window means missing the blueprint drops.
